# TrustWork

TrustWork is the premier Web3 freelance escrow platform built on the Stellar network. Designed to be the ultimate decentralized alternative to Web2 giants like Upwork and Fiverr, TrustWork eliminates 20% commission fees, prevents centralized account freezing, and introduces mathematically guaranteed smart contract escrows.

## The Web3 Gap: Solved

Traditional freelance platforms take massive cuts and hold all the power. TrustWork returns control to clients and freelancers through:

- **0% Protocol Fees**: Keep what you earn.
- **Smart Contract Escrow**: Clients lock XLM in a neutral smart contract upon hiring.
- **Automated Milestones**: Funds release instantly and automatically upon milestone approval.
- **On-Chain Arbitration**: Disputes are resolved by a decentralized jury of TRUST token holders.
- **TRUST Token**: Build immutable, portable on-chain reputation for every completed job.
- **AI Agent Intelligence**: Our integrated AI scores proposals and flags potential dispute patterns before they happen.

## Tech Stack

This project features a "Dark Luxury" premium user interface with buttery smooth scrolling and dynamic animations.

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) & [shadcn/ui](https://ui.shadcn.com/)
- **Animations**: [GSAP](https://gsap.com/) & ScrollTrigger
- **Smooth Scrolling**: [Lenis](https://lenis.studiofreight.com/)
- **Blockchain**: Stellar Network (via Freighter Wallet)

## Getting Started

### Prerequisites
- Node.js (v18+)
- [pnpm](https://pnpm.io/)
- [Freighter Wallet Extension](https://www.freighter.app/)

### Installation

1. Install dependencies:
```bash
pnpm install
```

2. Run the development server:
```bash
pnpm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to experience the platform.
